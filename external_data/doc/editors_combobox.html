Combobox editor represents a drop-down list allowing the user to select a value from this list. Available options can be either hardcoded or retrieved from a lookup data source.

Key properties:

MFU values
Most frequently used (MFU) values to be always displayed at the top of the drop-down list.
Item caption template
Allows to populate the editor with values of multiple columns from a lookup data source

Open Insert or Edit forms to see the editor in action.
