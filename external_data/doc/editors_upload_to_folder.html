Upload file to folder and Upload image to folder editors allow users to upload a file or an image to a specified folder.

Common properties:

Folder to upload
Specifies the folder to be used to store the uploaded files.
Store file name only
Defines whether the full file path or only the file name is stored in the database after the uploading.
Replace file if exists
Replaces file with the same name.

Open Insert or Edit forms to see these editors in action.
