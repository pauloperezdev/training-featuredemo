HTML Wysiwyg editor allows users to edit HTML content without having to type or remember names of HTML tags and attributes.

The editor was significantly upgraded in the version 20.5 and it goes now with the following new features:

  • The possibility of picking font families;
  • Font size changing;
  • Adding images in base64;
  • Table management;
  • Adding special characters;
  • Using of predefined HTML templates;

Open Insert or Edit forms to see the editor in action.
